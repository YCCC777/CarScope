Privacy Policy

Last updated: April 2026

CarScope does not collect, store on any server, or transmit any personal data. Risa Studio does not operate any server or database. All data stays on your own device.

What CarScope reads

When you visit a CarGurus vehicle listing page, CarScope reads the following publicly visible information from the page:

  • Vehicle details: make, model, year, trim, VIN, price, mileage
  • Dealer name, location, and phone number (as displayed on the listing)
  • Deal rating and fuel economy figures (as provided by CarGurus)

This information is identical to what any visitor can see on the listing page.

Where data is stored

All data is stored only in your own browser:

  • chrome.storage.local (persistent) — saved cars, notes, ratings, price history, browsing history, settings
  • chrome.storage.session (browser session only) — the compare list, cleared when the browser closes

Risa Studio and no third party can access this data.

External network requests

To provide safety and vehicle data, CarScope makes requests to the following free public APIs when you open a supported listing:

  • api.nhtsa.gov — NHTSA Recalls API: queries open recalls by make / model / year
  • api.nhtsa.gov — NHTSA Complaints API: queries user complaints by make / model / year
  • vpic.nhtsa.dot.gov — NHTSA vPIC VIN Decode API: decodes the 17-character VIN to vehicle specs

These requests send only vehicle make / model / year or VIN — no personal identifiers. Results are processed locally and not stored beyond the popup session (except the recall count saved to your local car record).

Background price refresh opens temporary browser tabs to re-read listing prices from CarGurus. These tabs are opened and closed automatically and are not visible unless your browser is configured to show them.

Permissions explained

CarScope requests the following Chrome permissions:

  • activeTab / tabs — reads the current tab's URL and opens links from the extension popup
  • scripting — injects extraction code into CarGurus pages (in the MAIN world) to read vehicle data from window.__remixContext
  • storage — saves your car list, notes, and settings to chrome.storage.local
  • alarms — schedules the 24-hour background price refresh
  • notifications — shows a system notification when a saved car's price drops (can be disabled in Settings)
  • host_permissions (cargurus.com, nhtsa.gov, etc.) — required by Chrome MV3 to inject scripts and make cross-origin requests to those domains; no data is sent to Risa Studio

No tracking or analytics

CarScope contains no analytics SDK, crash reporting service, advertising network, or telemetry of any kind. Risa Studio has no visibility into how many people use the extension, how often, or what vehicles they look at.

Children's privacy

CarScope does not collect any user information. Since data never leaves your device, there are no privacy concerns for users of any age.

Policy changes

If CarScope's privacy practices change (for example, a new data source is added), this policy will be updated and the extension version number will be bumped accordingly.
